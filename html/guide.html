User guide

The main screen shows the site area -big image displaying the current site. The object area is displayed at the bottom, it contains 6 pockets to carry objects, gray pockets are empty. When you enter a new location, the location name is shown in the top-left corner of the site area.

At the bottom-right corner there is a button to access the plant map. Your current place is marked with a big red star. Moreover, when you pick up an object, it is also shown with an 'x'.

You can interact in several ways through the game. To make it easier, sometimes a blinking star is shown in the site area, meaning that you can interact in that specific place.

Yellow starts (left) allow you to pick an object -if you have empty pockets it will be displayed in the object area-, red stars (center) allow you to use an object -see below to learn how to use an object- and, finally, green starts (right) allow to interact in a few specific places. If you move the mouse near a star, a big rectangle matching the star color is shown.

If you click an object in the object area, its description is shown. First, an enhanced object image, then the complete description and, finally, some buttons: "Return" to return to the game, "Use" to use the object and "Discard" to discard the object. Some objects cannot be discarded -the last button is not shown-.

To use an object in a red-star location, go to its description and click on the "Use" button. Now a red shadow is displayed around the object in the object area, meaning that it can be used in a red-star location; finally click in a red-star location.

You can interact also with people and, sometimes, with big locations. If you move the mouse over a person -and sometimes big locations-, you will see a green rectangle, if you click on it, you'll see his/her message at the bottom of the site area.

You can move to a different site using the blinking arrows that are shown in the site area, sometimes, you must use an object to move to another location. Normally, these arrows are pink, but blue arrows indicate the way -site- you just came.

That's all you need to know. Have fun!
